# Warp Prism


## How to use
`$ npm install -g warp-prism`  

### Generate Common Data Structures & Unit Test File    
`$ warpin doublyLinkedList data-structures`  

`$ cd data-structures`  

`$ warpin trie .`  


### Generate Custom Objects, specify directory later
`$ warpin zealot`  

`$ warpin adept`

`$ warpin highTemplar`

`$ cd psionic-matrix`

`$ ls -l`
